# HGS Backend

Hızlı Geçiş Sistemi (HGS) için geliştirilen, araç ve geçiş kayıtlarını yöneten Spring Boot REST API'si. Staj kapsamında geliştirilmektedir, React ön yüz entegrasyonu planlanıyor.


## Teknolojiler

- Java 21, Spring Boot 3.3.4
- Spring Data JPA + PostgreSQL
- Lombok
- spring-boot-starter-validation
